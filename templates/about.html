About

This website was created by Tiago Lubiana in 2024. It is designed to provide an easy-to-navigate interface for exploring the Vickaryous & Hall cell classification system using data from Wikidata and Wikipedia.

The site leverages the power of linked data and semantic web technologies to dynamically present information about various cell types and their hierarchical classifications.

For more information, you can visit my GitHub profile.
